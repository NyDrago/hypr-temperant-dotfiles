# hypr-temperant-dotfiles

my dotfiles i use that i made and am posting bc i need to contribute more

## Components
* **Widgets/Bar:** Eww
* **Notifications:** Swaync
* **Status Bar Alternative:** Waybar
* MAKE SURE TO DOWNLOAD THESE!!
* **Custom Scripts:** Included for system notifications and automation (they come in the install.sh just run that in your terminal)

NOTE: make sure to have swaync and ~/.config/hypr/scripts/media-notifier.sh as exec-once/startup in your hyprland conf

and if you want to be able to manually open the media player put ~/.config/hypr/scripts/media-notifier.sh
as a bind

## Installation
To install these configurations automatically, clone the repository and run the setup script:

```bash
git clone [https://github.com/NyDrago/hypr-temperant-dotfiles.git](https://github.com/NyDrago/hypr-temperant-dotfiles.git)
cd hypr-temperant-dotfiles
./install.sh
